# Development Environment (Tech Stack)
## Core Framework & Runtime
- Runtime: .NET 10.0 (Cross-Platform)
- Primary Framework: ASP.NET Core MVC (Model-View-Controller)
- UI Engine: Razor Pages (Server-side rendering)
## Programming Languages
- Backend: C# 13
- Frontend: HTML5, CSS3, JavaScript (ES6+)
- Scripting/Automation: TypeScript (Utilized for automated testing)
## Data & Persistence
- Database: SQLite (Relational, file-based for local development and portability)
- ORM: Microsoft Entity Framework Core (Code-First approach)
- Storage: Local File System (Abstracted via Service for image uploads)
## DevOps
- Containerization: Docker
- Automation: GitHub Actions
## Testing & Quality Assurance
- Functional Testing: Playwright (Browser automation for UI/UX validation)
- Load/Performance Testing: Locust (Python-based distributed user scaling)
## Development Tools & Infrastructure
- IDE: Visual Studio Code with C# Dev Kit
- Version Control: Git
- Architecture: x86-64
- Compatibility: Cross-Platform (Windows, Linux, macOS)
